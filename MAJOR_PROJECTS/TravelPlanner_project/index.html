Integrations & API Hooks

Suggested endpoints to practice with:
  • Places API (Mapbox / Here / Google Places) — search & autocomplete
  • Geocoding API — coordinates ↔ address
  • Directions API — route distances & durations
  • Weather API (OpenWeatherMap) — forecast by coordinate & date
  • Image APIs (Unsplash) — fetch photos for a place
All calls should be done from JS with fetch() and proper error handling. Practice building wrapper functions for each API.
